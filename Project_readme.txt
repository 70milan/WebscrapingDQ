-- Webscrapping using Beautiful Soup and csv to fetch amazon product data 
   and store it in Dataframe using Pandas.
	attributes:
	
                 product_name, product_description, ratings, ASIN number, Price
	
-- This dataframe can then be exported to PowerBI to write a "dynamic query".
-- This "dynamic query" enables the user to fetch (product_name, product_description, ratings, ASIN number, Price) attributes
   for any AMAZON URL entered in the "dynamic query"
